Part 2: Relevant Features

1. Spot Light/ or other Emitters: There will be at least one emitter in the scene. And this light could be set with different parameters to see a different effect of the scene.

2. Path tracing diffusive surface: For the pure diffuse surface in the scene, the path tracing could be used directly.

3. BRDF sampling: For the carbon object, microfaced model is necessory to used in the brdf sampling.

3. Photon Mapping: For the diamond object, photon mapping could be used for caustic effects and finally re using photon mapping for global illumination.
